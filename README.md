# TechNova Electronics Store

This project is a modern responsive website for an online electronics store.

## Concept
The website presents an online store where users can browse electronics products, categories and special deals.

## Features
- Hero section with promotional banner
- Popular categories
- Best-selling products
- Call-to-action buttons
- Responsive mobile layout

## How to run the project
1. Download the repository
2. Open index.html in a browser
