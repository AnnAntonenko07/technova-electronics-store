# TechNova Electronics Store

A modern, responsive e-commerce website for an online electronics store built with React and Vite.

## Concept
The website presents an online store where users can browse electronics products, categories, and special deals with a clean, professional design.

## Features
- Responsive design (desktop and mobile)
- Hero section with promotional banner
- Featured products showcase
- Popular categories grid
- Trust badges section
- Newsletter subscription
- Contact form
- Multi-page navigation (Home, Shop, About, Contact)

## Tech Stack
- **React 18** - UI Framework
- **Vite 6** - Build Tool
- **React Router 7** - Navigation
- **Tailwind CSS 3** - Styling
- **Lucide React** - Icons

## Project Structure
```
/src
  /components    # Reusable UI components
  /pages         # Page components
  /styles        # CSS styles
```

## How to Run the Project

### Prerequisites
- Node.js 18+ installed

### Installation
```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Pages
- **Home** - Hero banner, featured products, categories, trust badges
- **Shop** - Product listing with filters and sorting
- **About** - Company information and values
- **Contact** - Contact form and business information
